## Movies API

Clone the repo, run ``npm start`` and you will see a search bar where you can type any movie title and a list of movies will pop up. 
To check the details, click on ``details`` button.

USED API: http://www.omdbapi.com/

USED  TECH STACK:
- ``React``
- ``Axios``
- ``React-bootstrap``
- ``React-router-dom"``
